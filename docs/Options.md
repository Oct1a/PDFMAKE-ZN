选项:

`fontLayoutCache` - 最低版本: **0.1.55**

- `fontLayoutCache` - 禁用字体布局缓存的选项 (```default: true ``` - 启用缓存)
- `bufferPages` - 允许你控制何时将页面刷新到输出文件。 ([查看文档](http://pdfkit.org/docs/getting_started.html)) (```default: false``` - 禁用缓冲区)
